# Employee & Recruiter’s Helping Dashboard 📊

A comprehensive data visualization dashboard designed to assist **employees** and **recruiters** in analyzing job market trends, salary distributions, work formats, and country-specific insights. Built for clarity and accessibility, this dashboard empowers decision-making with interactive charts and filters.

## 🛠️ Tech Stack

- **Data Source**: Employment & recruiter datasets
- **Visualization Tools**: Power BI / Excel / Python (Pandas, Matplotlib)
- **Interactivity**: Filters for job title, work format, country, and salary ranges

---

## 📂 Dashboard Highlights

- **Filters Sidebar**: Select job titles, work format (On Site / WFH), countries, and salary ranges.
- **Pie Chart**: Work format distribution.
- **Bar Charts**: 
  - Salary distribution by range
  - Average salary by country
  - Yearly salary by job title

---

## 🎯 Use Cases

- **Recruiters**: Benchmark salaries across roles and regions.
- **Employees**: Compare compensation trends for career planning.
- **Organizations**: Analyze work format preferences (WFH vs On Site).
- **Data Analysts**: Explore structured employment datasets for insights.
